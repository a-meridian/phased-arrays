Tier Description Criteria
Tier 1 High Quality Peer-reviewed, replicated, large sample, rigorous methodology
Tier 2 Moderate Quality Published research, limited replication, methodological concerns

1.2 Confidence Levels

Level Description Interpretation
High Strong evidence base Phenomenon likely exists as described
Medium Mixed evidence Phenomenon possible, mechanism uncertain

5. Recommended Evidence Priorities

5.1 High Priority (Would Substantially Increase Confidence)

  1. Coherence measurement methodology: Develop and validate population-level r estimation
  2. Pre-registered collective coherence studies: Independent replication with pre-registration
  3. Mechanism research: How would collective coherence produce measurable effects?
  4. Threshold observation: Document if/when threshold dynamics occur

5.2 Medium Priority

  1. Injection locking dynamics: Quantify belief capture under controlled conditions
  2. Practice effects dose-response: Map G_practices to specific interventions
  3. Paradigm shielding disaggregation: Measure relative contribution of each component
  4. DNA activation correlates: Identify biological markers

5.3 Lower Priority (Difficult to Address)

  1. Source signal validation: Inherently unfalsifiable by physical methods
  2. Path loss estimation: Would require cross-density measurement
  3. Ratchet mechanism: Would require long-term developmental tracking
